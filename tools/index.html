1: Git, GitHub, and Markdown

These are all essential skills in the modern developer toolbox.

  • Git is our version control system. It's abundant in use in the industry and knowing it well is vital for collaborating and writing code. Readup!
  • GitHub is a central Git-server which makes collaborating with code easy. We use it to store all our code. GitHub hosts an impressive amount of open source software and is something you should be intimately familiar with! Check out our organisation.
  • Markdown is the go to markup language for authoring documentation, comments, websites, and more. Markdown let's you write documents in plain text without worrying about the formatting - great for productivity! It's another core tool that you want to make friends with ðŸ˜„
